## Requirements
* Node.js
* React Native CLI
* Jest

--------------------

## Run
1. Download and install the Android Studio
2. Open the emulator
3. Run `npm install`
4. Run `npm run dev`

--------------------

## Test
* Test: `npm test`
* Test watch: `npm run test:watch`
* Test coverage: `npm run test:coverage`
* Linter: `npm run eslint`

--------------------

## Environments variables
* .env
* .env.staging
* .env.production

--------------------

## Build
* Build Android: `npm run build:android`

--------------------
